### Ubuntu Server Setup
- Install The Regular Ubuntu LTS
- Go to Settings -> Sharing -> Remote Desktop 
- Activate And Done!

### Dconf editor
For editing the annoying promt <br>
Change promt to password after installing dconf editor <br>
`sudo apt install dconf-editor` <br>
